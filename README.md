# Proyek Analis Data 
## Kualitas Udara

## Setup Enviroment
```
conda create --name main-ds python=3.9
conda activate main-ds
pip install -r requirements.txt
```
## Setup Environment - Shell/Terminal
```
mkdir analisis_data
cd analisis_data
pipenv install
pipenv shell
pip install -r requirements.txt
```
## Setup Dashboard
```
cd Dashboard
```
## Run steamlit app
```
streamlit run dashboard.py
```
